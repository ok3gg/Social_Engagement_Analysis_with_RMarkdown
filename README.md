# Social_Engagement_Analysis_with_RMarkdown

This report explores patterns in social and volunteering engagement in Singapore. The dataset of 3076 respondents used contains survey results from the MCCY (Ministry of Culture, Community and Youth) Social Values Survey on attitudes and actions that inform Singaporeans' desire to build social capital.

Files for survey data and description of variables have been modified (certain columns removed).

Dataset Source: https://data.gov.sg/collections/397/view
